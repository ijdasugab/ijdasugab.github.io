# Bagus Samudra Portfolio

Personal portfolio website for GitHub Pages.

## Structure

- `index.html` - main portfolio content.
- `style.css` - responsive layout and visual styling.
- `script.js` - automatic copyright year.
- `favicon.svg` - site icon.

## Deploy to GitHub Pages

1. Create a new GitHub repository named `ijdasugab.github.io`.
2. Push the files in this folder to that repository.
3. Open `Settings > Pages`.
4. Select the `main` branch and the `/root` folder.
5. Save the settings, then open `https://ijdasugab.github.io`.

## Customization

Update the profile text and social links in `index.html`.
